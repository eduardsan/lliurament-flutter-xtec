# lliurament_flutter_xtec

Aquest projecte és una aplicació Flutter desenvolupada com a projecte educatiu per al curs de Desenvolupament d'Aplicacions Multiplataforma amb Flutter de l'XTEC. Implementa una arquitectura MVVM amb Clean Architecture, utilitza la llibreria Drift per a la persistència de dades en una base de dades local SQLite, i permet gestionar (afegir, editar, eliminar) targetes de Pokémon amb imatges i dades obtingudes de PokeAPI.
